Easy COVID-19 Map

 

The Easy COVID-19 Map visualises the number of confirmed COVID-19 cases, as well as the number of recovered people and deceased people per million inhabitants according to the John Hopkins Coronavirus Resource Centre.  What’s more, the map visualizes the growth rate at which the virus is spreading in each country giving the user a much clearer perspective of the current situation.

 

The map updates itself every time the user access the site and gives an overview over time. The user can also see the development of the pandemic as of January 22nd, 2020.

 

Project originally developed for the #WirVsVirus Hackathon held on 20/03/2020.

 

Stay updated on Devpost: https://devpost.com/software/easy-covid-19-map
